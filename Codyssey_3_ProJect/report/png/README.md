# report/png — n8n 설치·OAuth 마찰 미디어

확장자가 `.png` / 생성 스크립트·안내 문서를 이 폴더에 모은다.  
(Make 실행 GIF는 `report/gifs/` 에 둔다.)

## 포함 파일

| 파일 | 설명 |
|------|------|
| `_render_setup_friction.py` | 맑은 고딕(malgun.ttf)으로 패널 재생성 |
| `n8n_setup_or_oauth.gif` / `.png` | 6장면 스토리보드 |
| `n8n_friction_01` ~ `06` `.png` | 장면별 정지 이미지 |

## 재생성

```text
py -3.12 Codyssey_3_ProJect/report/png/_render_setup_friction.py
```

스크립트 출력 경로: **이 폴더(png/)**.
