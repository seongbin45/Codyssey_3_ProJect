## [프로젝트 1] 자동화 도구 비교 구현

- `[ ]` **1단계. 데이터 소스 준비**
  - `[ ]` 구글 폼 생성 (질문: 지출 메모 1개)
  - `[ ]` 구글 시트 생성 (헤더 6열: 타임스탬프, 원본 메모, 카테고리, 금액, 요약, 특이사항)

- `[ ]` **2단계. 도구 A — Make.com 워크플로우 조립**
  - `[ ]` Make.com 회원가입 및 시나리오 생성
  - `[ ]` Google Forms 트리거(Watch Responses) 연결
  - `[ ]` OpenAI(ChatGPT) 모듈 연결 (System Prompt + JSON Mode)
  - `[ ]` Router 추가 → 3-Way 분기 조건식 입력
  - `[ ]` 각 분기별 Google Sheets(Add a Row) 매핑
  - `[ ]` 분기 A(고액) + 분기 C(예외)에 Slack 알림 매핑
  - `[ ]` 테스트 3종 실행 (고액 / 소액 / 예외)
  - `[ ]` 워크플로우 구성 화면 스크린샷 캡처
  - `[ ]` 실행 결과 화면 스크린샷 캡처 (각 분기 1회 이상)

- `[ ]` **3단계. 도구 B — Zapier 워크플로우 조립**
  - `[ ]` Zapier 회원가입 및 Zap 생성
  - `[ ]` Google Forms 트리거 연결
  - `[ ]` ChatGPT 모듈 연결 (동일 프롬프트)
  - `[ ]` Paths 추가 → 3개 Path 조건식 입력
  - `[ ]` 각 Path 내 Google Sheets + Slack 액션 배치
  - `[ ]` 테스트 3종 실행
  - `[ ]` 워크플로우 구성 + 실행 결과 스크린샷 캡처

- `[ ]` **4단계. 프로젝트 1 비교 분석 보고서 작성**
  - `[ ]` 보고서 마크다운 파일 생성 (제출용)
  - `[ ]` 필수 항목 5개 포함 확인
  - `[ ]` 스크린샷 삽입
  - `[ ]` 보안 마스킹 처리 확인
